# 🧩 Sudoku Validator (Java Project)

## 📌 Overview
This project validates whether a given 9x9 Sudoku board follows standard Sudoku rules. It ensures that each row, column, and 3x3 sub-grid contains unique digits from 1 to 9.

## 🎯 Objective
To implement a logic-based validation system using Java concepts such as 2D arrays, loops, and HashSet.

## ⚙️ Features
- Row validation
- Column validation
- 3x3 grid validation
- Duplicate detection with meaningful messages
- Clean console output

## 🧠 Concepts Used
- 2D Arrays
- Nested Loops
- HashSet (for duplicate checking)
- Conditional Statements

## ▶️ How to Run
1. Compile the program:
   javac Main.java

2. Run the program:
   java Main

## 📊 Output
The program displays the Sudoku board and checks whether it is valid or not.
